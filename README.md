## 데이터형
- 데이터형이란 데이터의 종류

### JavaScript의 주요 데이터형
- 기본형
  * 숫자형(number)
  * 문자열형(string)
  * 논리형(boolean)
  * 심벌형(symbol)
  * 특수형(null/undefined)
- 참조형
  * 배열(array)
  * 객체(object)
  * 함수(function)

***값을 변수에 대입하는 방법에 차이점이 있다.
기본형 : 변수에 값 그 자체가 직접보관
참조형 : 변수에 참조값(값을 실제로 보관하고 있는 메모리의 어드레스) 보관***
